# Node.js Express을 활용한 커피숍 등록/조회 프로그램
## 사용Skills: 
####         1. Node.js , Express
####         2. View : BootStrap, Semantic 
####         3. 데이터베이스 :MongoDB
####         4. 기타: 구글 Map API

## 구현기능: 
####         1. 페이스북으로 로그인해야만 사용이가능하다 
####         2. 사용자는 구글맵 API를 이용하여 원하는 위치에 커피숍을 등록할수있다.
####         3. 등록한 커피숍 및 근처에있는 커피숍의 정보를 조회할수있다.
